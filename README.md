# LinkedListExample

Question:
Given a non-empty, singly linked list with head node, return a middle node of linked list. If there are two middle nodes, return the second middle node. 

Example:
> Input: [1, 2, 3, 4, 5, 6]
> Output: 4 

``` c=1
struct Node
{
    int val;
    struct Node *next; 
};
```
